# myntra-clone
This is a simple Myntra clone project created using *HTML and CSS*.
The purpose of this project is to practice frontend development skills
and understand website layout and styling.

##  Features
- Homepage layout similar to Myntra
- Navigation bar
- Product sections
- Clean and responsive design

##  Technologies Used
- HTML5
- CSS3

##  Project Structure
- index.html
- style.css

## What I Learned
- HTML page structure
- CSS styling and layout
- Flexbox basics
- Hosting website using GitHub Pages
